# FlightOps Commander

Plataforma profissional de operações aéreas - v0.1.0

## Funcionalidades

- **Dashboard**: Visão geral da missão em tempo real
- **Checklists**: Procedimentos estruturados e verificações
- **Combustível**: Cálculos de consumo e autonomia
- **Planeamento**: Planeamento e briefing de missão
- **PWA**: Funciona offline e instalável como app

## Tecnologia

- Next.js 15
- TypeScript
- Tailwind CSS
- Framer Motion
- Recharts
- Lucide Icons

## Setup Local

```bash
npm install
npm run dev
```

Aceda a http://localhost:3000
